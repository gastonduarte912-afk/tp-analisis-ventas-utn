## Escenario elegido
Escenario B - Análisis de Ventas de una Pequeña Empresa

## Objetivo
Analizar datos de ventas utilizando Python, Git y GitHub para generar estadísticas y gráficos automáticos.

## Descripción del proyecto
Este proyecto permite procesar información de ventas comerciales utilizando herramientas de análisis de datos.

El sistema:
- importa datos desde un archivo CSV
- calcula ventas totales
- detecta productos más vendidos
- genera gráficos automáticamente
- guarda resultados en archivos

## Tecnologías utilizadas

- Matplotlib
- Git
